SAY HI TO THIS CUTE BOT THAT WON'T STOP COMPLIMENTING ME

Node.JS / Twilio API / OpenAI API Built in 2025

Let me explain

View transcript A day before my birthday, I decided to build a WhatsApp bot that would send me a compliment every hour on the day.

I built this using a Node.js script, scheduled with GitHub Actions to run every hour. Each run picks a random entry from a list of personal achievements I put together and sends it to the Gemini API to generate a compliment. The compliment is then delivered it to me via WhatsApp using Twilio.

To make it more amusing, I paired some of the achievements with images, so I’d get a message that included both the compliment and a photo tied to that moment.

If you’re curious to build one of your own, the code is up on my GitHub.
