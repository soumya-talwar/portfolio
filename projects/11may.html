SAY HI TO THIS CUTE BOT THAT WON'T STOP COMPLIMENTING ME

Node.JS / Twilio API / OpenAI API Built in 2025

Let me explain

View transcript Birthdays are usually a sad affair for me. But this year, a day before my birthday, I decided to program a WhatsApp bot that would send me a compliment every hour on the day itself. Because nothing quite says “emotionally intelligent” like automating yourself some self-love.

I built this bot using the OpenAI API and Twilio, running on a Node.js environment. Basically, my code would ping ChatGPT every hour and deliver whatever compliment ChatGPT generates straight into my WhatsApp.

To make it a little more personal, I spent some time reflecting on past wins and moments that I was proud of—and paired those with a few photographs. And with that context, ChatGPT started sending compliments that felt surprisingly sincere... and only mildly, suspiciously specific.

If you’re curious, the code is up on my GitHub—feel free to build your own birthday bot with it.
