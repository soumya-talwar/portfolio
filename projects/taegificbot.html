TAEGI
FIC BOT

[ visit github code ]

Nobody knows the struggle of finding good queer fanfiction like me. Which is why I built @taegificbot, a queer BTS fanfiction recommendation engine that gave instant customised fanfiction recommendations to Twitter users.

It started with a simple tweet wishing for a way to get instant fanfic recommendations instead of scraping all of Archive of Our Own every night and finding nothing. Responses from other despondent Twitter users poured in and they were the encouragement I needed. I was motivated to build a solution and together with two other fanfic enthusiasts, I was able to populate a spreadsheet with thousands of links to fanworks we'd all read and loved in the fandom. Since Archive of Our Own has no API to request data from, I built a web scraper that pulled all the metadata of a story when given its url. All of that data was then stored in a realtime database on Firebase.

Next, I built a Twitter bot that would be triggered each time a user mentioned the bot's username in a tweet and typed "recommend". The bot would then pull links from the database and reply to the user. These recommendations could be customised as well. Users could request for stories with specific word counts, pairings, genres, tropes, completion statuses, warnings, etc. Moreover, they could subscribe to receive daily fanfic recommendations in their inbox.

Unfortunately, I had to take down the Twitter bot a few months after its launch due to both personal reasons and some negativity I received on the platform.
